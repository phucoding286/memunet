# Explain (Giải thích).
> MEMUNET là một mã code định danh dành cho các tools có mã này, các tools có đánh dấu mã này có nghĩa là các tools ấy chia sẻ thông tin chung với nhau để tối ưu hóa hành vi của nó và dùng chung tài nguyên các máy ảo Memu để tận dụng tối đa tài nguyên có sẵn mà không cần tạo quá nhiều VMs gây lãng phí.

# Lưu ý khi chạy các tools thuộc mã định danh MEMUNET.
1. Nếu 1 tool tắt đi thì buộc các tools khác cũng phải tắt theo, nếu không sẽ gặp lỗi treo.
2. Tuy nhiên ở phần bật, bạn có thể bật bất kỳ tools nào mà không cần bật hết tất cả cùng lúc.
